### ◎函数

1. 函数两个基本要素：定义域和对应规则（或称依赖关系）。当两个函数的定义域与对应规则完全相同的时候，他们就是同一个函数。  
Q：为什么不是三个？值域呢？  
A：其实确定了定义域与对应规则值域自然也确定了，所以只有两个基本要素</font>

2. 常见函数有符号函数、取整函数、复合函数、反函数、初等函数。
	+ 取整函数的定义（<u>不超过x的最大整数</u>）。  
	用例子去理解：`3.2=3+0.2`, `-3.2=-4+0.8`, 所以 `[3.2]=3`,`[-3.2]=4`。
	+ 不是任意函数都能复合，条件是**外层函数定义域与内层函数值域的交集为非空**时才可以。  
	比如说 `f(u)=lnu` 和 `u=g(x)=sinx-1` 不能复合。
	+ 不是每一个函数都有反函数。  
	比如 <code>y=x<sup>3</sup></code> 有反函数,但是 <code>y=x<sup>2</sup></code> 没有反函数。  
	有反函数等价于 ∀x<sub>1</sub>≠x<sub>2</sub>∈D,有f(x<sub>1</sub>)≠f(x<sub>2</sub>)。
	+ 单调函数一定有反函数，但反之则不然。比如一些“分段函数错开分界点的函数”。
	+ 如果有反函数，则图形上，函数与他的反函数关于 y=x 这条直线对称。
	+ f<sup> -1</sup>[f(x)]=x , f[f<sup> -1</sup>(x)]=x。  
	意思就是函数与他的反函数复合，结果都是"x"。这叫做**恒等映射**。
	+ 基本初等函数有幂函数、指数函数、对数函数、三角函数以及反三角函数。 （！清楚了解其定义域、性质及其图形）。
	+ 初等函数的定义：由常数和基本初等函数经过**有限次四则运算**和**有限次的函数复合**所构成并**可用一个式子**表示的函数，称为初等函数。  
<br>

3. 函数的性质有：单调性、奇偶性、周期性、**有界性**。
	+ --><mark>奇偶性</mark>
	+ 奇偶性值得注意的是，如果函数定义域关于原点不对称，那么根本谈不上奇偶性。
	+ 常见奇函数,常见偶函数在复习全书基础篇P6 和 A4复习计划P2有，记得有空看看。
	+ 奇函数图形关于原点对称，且只有在x=0处有定义的时候，f(0)才等于0。
	+ 奇+奇=奇，偶+偶=偶，奇x奇=偶，偶x偶=偶，奇x偶=奇。
	+ --><mark>周期性</mark>
	+ 周期性中熟悉常见基本初等函数的周期。
	+ --><mark>有界性</mark>
	+ 有界函数的定义：设y=f(x)在集合X上有定义。若存在M>0，使得对任意的x∈ X，恒有|f(x)|≤M，则称f(x)在X上为有界函数。
	+ 无界函数的定义：如果对任意的M>0，至少存在一个x<sub>0</sub>∈X，使得|f(x<sub>0</sub>)|>M，则称f(x)在X上为无界函数。
	+ 有界性不能笼统的去讲，具体要说明这个函数在哪里有界，哪里无界。如果无说明，则认为在其定义域中为有界函数。  
	（拿 `y=1/x` 作为例子理解，在(0,1)无界，在(1,2)有界。）

<br>
	
4. 关于证明无界函数的问题，就是运用无界函数的定义，找到这个x<sub>0</sub>，使得|f(x<sub>0</sub>)|大于任意的M(>0)即可。具体例子可以参考 <u>复习全书基础篇P7 例5。</u>

#### 极限的概念
1. 数列的极限
	+ 定义：要理解！<mark>∀ε>0,∃N>0,当n>N时，恒有|X<sub>n</sub>-a|<ε,则称常数a为数列{X<sub>n</sub>}当n趋向于无穷时的极限记作<img src="https://bkimg.cdn.bcebos.com/formula/d8b8dde843d5675f9d0de0eb02cb314d.svg"></mark>
	+ ε的作用：刻画数列项（X<sub>n</sub>）与这个常数a（极限值）的接近程度。
	+ ε的任意性，做一些题目时有用。ε可以任意取，可以有多小取多小。
	+ N的作用：用来刻画n趋向于∞这个极限过程。
	+ <img src="https://bkimg.cdn.bcebos.com/formula/d8b8dde843d5675f9d0de0eb02cb314d.svg"> 的几何意义！
	+ 数列{X<sub>n</sub>}的极限是否存在，如果存在极限值等于多少与数列的前有限项无关，与后面无穷项有关，因为数列极限研究的是
	n无穷增大时的变化趋势。
	+ 如果一个数列极限为a，那么他的奇数列与偶数列极限也为a。但是反过来，只有一个数列的奇数列和偶数列的极限存在且相等时，
	才可以推出这个数列有极限。
	+ <u>复习全书基础篇P8 例1</u> 因为出现 (-1)<sup>n</sup>，从而很自然进行n是奇数还是偶数的考虑，从而也引出了这道题的考点：
	"奇偶数列极限推数列极限"。即只有奇偶数列极限存在且相等时，才可以推出这个数列有极限。
	+ <u>复习全书基础篇P8 例2</u>这道证明题让我们充分理解数列极限的定义！概念题是考生的薄弱项目，多体会这道题。  
	如何从|X<sub>n</sub>-a|<ε 推出 ||X<sub>n</sub>|-|a||<ε 是关键！这里涉及到绝对值不等式的知识，稍微总结一下。特别说明一下，定理2是基本不等式，定理5是这道题所要用到的是绝对值三角不等式。

2. 函数的极限  
1）自变量趋向于无穷大时函数的极限
+ 自变量趋向于∞时函数的极限存在的充要条件是趋向于-∞与+∞时的极限存在且相等。
+ <u>复习全书基础篇P9 例3</u> 一个经典错误是 误将√x<sup>2</sup>以为是=x。这个题有两个解法，第一就是把x<sup>2</sup>开出来
，第二种解法就是把分母的x凑进去。但是这两种解法都要分情况讨论，因为题目中只是给出了x->∞,所以要分-∞和+∞讨论，最后运用上面的定理，
+∞和-∞存在且相等的时候才能推出∞极限存在。  
<br>
2）自变量趋向于有限值时函数的极限
+ 自变量趋向于 x<sub>0</sub> 极限存在的充要条件是左极限(x->x<sub>0+</sub>)及右极限(x->x<sub>0-</sub>)存在且相等。
+ 需要分左右极限求极限的问题有：  
**◎分段函数在分界点处的极限**（在该分界点两侧函数的表达式不同）。  
◎**e<sup>∞</sup>型**极限  
见到e<sup>∞</sup>型极限 ，不能盲目下结论，要看e指数，e<sup>+∞</sup> = +∞,e<sup>-∞</sup> = -∞。  
-->例如<u>复习全书基础篇P10 例4</u>，解题关键在于第一反应看到x->1时，出现了e<sup>∞</sup>型极限，就要第一反应分左右极限处理求出左右极限。然后按照自变量自变量趋向于有限值时函数的极限的定理，判断这一点左右极限
是否存在且相等判断原极限是否存在（存在则求其值）。  
◎**arctan∞型**极限  
见到arctan∞型，不能盲目下结论，arctan(+∞)=Π/2，arctan(-∞)=-Π/2。


##### 极限的性质
1. 有界性  
数列：如果数列{X<sub>n</sub>}收敛，那么数列{X<sub>n</sub>}一定有界。  
（如果一个数列收敛，那么这个数列一定有界。但是反过来就不一定，即数列有界不一定收敛，例子为 X<sub>n</sub> = (-1)<sup>n</sup>）。  
（无界数列一定发散，但发散数列不一定有界）。  
函数：若f(x)当x->x<sub>0</sub>存在，则f(x)在x<sub>0</sub>某去心领域有界（即局部有界）。  
（反过来不成立，比如说f(x)=sin1/x，该函数在x=0处的去心邻域有界，但它在x=0处的极限不存在（总在[-1,1]振荡，极限不存在）。）  
2. 保号性（这个很重要！，理解定义，概念！）  
◎数列  
1）**从极限值正负保数列项的正负**。  
如果A>0（或A<0)，则存在N>0，当n>N时，X<sub>n</sub>>0（或X<sub>n</sub><0）。  
<mark>怎么理解!</mark>：如果一个数列极限值A>0(A<0)。我不能保证你所有X<sub>n</sub>都大于0(或小于0)，但是一定有一个N，当n>N时，即从
某一个N以后，那些X<sub>n</sub>都大于0（或小于0）。  
<mark>为什么?</mark>因为一个数列的极限与前有限项是无关的，它反映的是后面无穷多项的变化趋势，也就是说它只管n充分大后的
项，前面管不了。（**数列极限反映的究竟是什么！**）    
2）**从数列项的正负保极限值的正负**。  
如果存在N>0，当n>N时，X<sub>n</sub>≥0(或X<sub>n</sub>≤0)，则A≥0（或A≤0）。   
<u>值得注意的细节</u>：数列保号性这两条性质可以简单归纳为  
A>0  --->  X<sub>n</sub>>0  
X<sub>n</sub>≥0 --->  A≥0   
可以看出第一个是严格不等号第二个是非严格不等号。那么第一个是否可以两边都加上等号，第二个是否可以去掉等号呢？ 不行。  
---> 第一个加上等号后，举反例 X<sub>n</sub>=(-1)<sup>n</sup>/n ,当n->∞时，极限A=0。如果按照第一条性质加上等号，A=0 --> X<sub>n</sub>=0，显然是不行的。
因为无论取多大，始终是奇数项在左边，偶数项在右边，一正一负，即不能保证n无穷大后面的项有X<sub>n</sub>=0。  
---> 第二个去掉等号后，举反例 X<sub>n</sub>=1/n，而 n->∞时，极限为0。 这就矛盾了，怎么矛盾了: X<sub>n</sub>都是大于0的，极限A却等于0。所以第二条还可以这么写
X<sub>n</sub>>0 --->  A≥0 。（右边加上等号，左边不加）。   
◎函数  
函数极限这边的保号性就不写出来。本质上或者应该说是同理。不同的是函数极限保号性保的是去心邻域，数列极限保号性保的是n充分大。且无论是有界性还是保号性都是这个理。自己体会！！  
推荐一篇文章--> [极限局部保号性的理解](https://lclfeng.github.io/2016/07/24/%E7%AC%AC%E4%B8%89%E7%AF%87/)  
--><u>复习全书基础篇P11 例5</u>是个很不错的题目。本题设计的知识点比较多，且都是概念知识点，有函数保号性的定义，极值定义，某一点导数定义。首先根据函数极限保号性，可以知道在自变量趋向于a
的时候，极限是等于-1（-1不重要，重要是<0）。所以f(x)在这个去心邻域也是<0的，所以这个函数表达式<0，表达式分母>0，整体小于0，所以f(x)-f(a)<0，从而f(x)<f(a)。代表在a点附近的f(x)的值都是小于a点函数值的，
这里就用到极值的定义，这就是极大值！。但关于选项中关于f(x)导数是否存在以及f'(a)的值是多少，还需要其他方法去求得。  
这类题也可以用排除法，关于排除法什么时候用，怎么用在这里说一下。当给出的函数时抽象函数的时候，即没有给出具体函数的时候可以用排除法，然后找符合这个抽象函数条件的最简单的
具体函数，比如说这道题中可以令f(x)就为-(x-a)<sup>2</sup>。为什么会想到这个具体函数？因为我们要求f'(a)，取-(x-a)<sup>2</sup>，f(a)就等于0了，f'(a)也为0了，另外前面的负号是因为
题目中要求极限为-1。用这个方法也能解释f(x)导数是否存在与f'(a)的值。  
还有一种方法就是涉及某点导数的定义，这里不方便细说，即找出a点导数的定义与原式的关系，从而判断出a点导数值。 

3. 极限值与无穷小的关系  
什么关系呢？用通俗的话来讲就是函数值和极限值之间仅仅相差一个无穷小。  
<u>lim f(x) = A 等价 f(x) = A + α(x)</u>

##### 极限的存在准则
1. 夹逼准则
2. 单调有界准则
	+ 单调有界数列必有极限。  
	单调增有上界必有极限（因为单调增已经不用考虑是否有下界，第一项就是最小的）  
	单调减有下界必有极限（因为单调减已经不用考虑是否有上界，第一项就是最大的）

这两个即数列存在性常用的两个准则，但每种方法都有适合的题型。  
+ 夹逼准则比较多的是用在n项和的数列极限。  
+ 单调有界准则比较多的是用在递推关系 X<sub>n+1</sub> = f(X<sub>n</sub>) 所定义的数列极限。而关于用单调有界准则解题步骤：
	+ 先用单调有界准则证明这个数列极限存在（关键！）
	+ 再根据 X<sub>n+1</sub> 与 f(X<sub>n</sub>)的关系，等式两边取极限，解代数方程即可。

【注：函数极限也有对应的两条存在准则，但考卷中存在准则还是更多用在数列极限。】

-><u>复习全书基础篇P12 例6</u> 这道题需要注意的是，不要认为 n->∞时每一项极限都趋向于0，所以加起来也趋向于0。要记住：**和的极限等于极限的和是针对有限项的**。（为什么？ 比如说这题每一项极限都趋向于0，那么无穷
项极限为0相加就相当于 0x∞型的极限，而这样的极限不一定存在，确切来说这是一个拉锯战，是一个未定式）。而分析题目可知是求n项和数列的极限，故使用夹逼准则更妥。  

-><u>复习全书基础篇P12 例7</u> 这道题首先涉及到取整函数的一个不等式即 x-1<[x]≤x，然后根据原式进行恰当的变形（同乘以x），再用夹逼准则即可解得。

-><u>复习全书基础篇P12 例8</u> 这道题比较明显的是使用单调有界准则。当然因为有f(X<sub>n</sub>)的表达式，所以X<sub>n+1</sub>与X<sub>n</sub>的关系很容易得出（X<sub>n+1</sub> = X<sub>n</sub> * 2/n+1 ），
又 X<sub>n+1</sub>与X<sub>n</sub>的极限是一样的都同设为a，则两边取极限即可求得a的值。  
值得一说的是：这道题其实还可以使用夹逼准则，只是夹逼的“左”和“右”并不是这么容易看的出来。这里不展开细说。

##### 无穷小量
1. 无穷小量的概念（什么叫无穷小量？）  
简单来说其实就是：极限为0的变量就叫无穷小量。0是唯一可以看作为无穷小量的一个常数。

2. 无穷小的比较
高阶、低阶、等阶、等价、无穷小的阶。  
-><u>复习全书基础篇P13 例9</u> 这道题考点就是无穷小的比较，简单粗暴一比就是，即 f(x)/x ,这里用到一个重要极限 lim(x->0) a<sup>x</sup>-1 / x = lna。

3. 无穷小的性质
+ 有限个无穷小的**和**仍是无穷小。（反例： lim<sub>n->∞</sub> (1/n<sup>2</sup> + 2/n<sup>2</sup> +... n/n<sup>2</sup> )）
+ 有限个无穷小的**积**仍是无穷小。 （不细说，相乘可以为1）
+ 无穷小量与有界量的**积**仍是无穷小。(lim<sub>n->∞</sub> xsin1/x，这个例子极限不能分开相乘，因为极限可以相乘的前提是极限都存在，lim<sub>n->∞</sub> sin1/x 极限不存在，
只能用无穷小量乘以有界变量等于无穷小量来说明这个极限为0)。


##### 无穷大量
1. 无穷大量的概念  
若对任意给定的M>0，总存在δ>0，当0<|x-x<sub>0</sub>|<δ时，恒有|f(x)| > M，则称f(x)为x->x<sub>0</sub>时的无穷大量。

2. 常用的一些无穷大量的比较 ！！  
1）函数：当x->+∞时， <mark>ln<sup>α</sup>x << X<sup>β</sup> << a<sup>x</sup></mark>，（其中 α>0，β>0,a>1）  
即当x->+∞时，后面一个比上前面一个，极限是无穷；前面一个比上后面一个极限为0。  
2）数列：当n->∞时， <mark>ln<sup>α</sup>n << n<sup>β</sup> << a<sup>n</sup> << n! << n<sup>n</sup></mark>，（其中 α>0，β>0,a>1）  
-><u>复习全书基础篇P14 例10</u> 这道题的考点就是无穷大量的比较，把上面常用无穷大量比较给理解了！

3. 无穷大量的性质
+ 两个（可以是有限个）无穷大量的**积**仍为无穷大量。  
（为什么∞+∞≠∞，因为说无穷大的时候是说他的绝对值 ->∞，即|f(x)| -> ∞，可见原无穷大可以是正可以是负，如果是一正一负就相互抵消，加起来就不是无穷大了，
所以正确可以这么写 (+∞) + (+∞) = +∞） 
+ 无穷大量与有界变量之**和**仍为无穷大量。  
(为什么 有界 x 无穷大 ≠ 无穷大？ 例子： 1/n * n = 1)。

4. 无穷大量与无界变量的关系(好好体会这两个概念的区别！)  
1）数列{X<sub>n</sub>}是无穷大量： ∀M>0，∃N>0，当n>N时，恒有|X<sub>n</sub>|>M。  
2）数列{X<sub>n</sub>}是无界变量： ∀M>0，∃N>0，使|X<sub>n</sub>|>M。   
不难看出：**无穷大量必为无界变量，而无界变量不一定是无穷大量。**  
（而对于无界x无界 =? 无界 这个问题，答案是不一定是无界的。不方便展开细说，A4复习计划P17有，详情可以看看。)

5. 无穷大量与无穷小量的关系  
在同一极限过程中，如果f(x)是无穷大，则 1/f(x)是无穷小；反之，如果f(x)是无穷小，**且f(x)≠0**，则 1/f(x)是无穷大。  
(等于0，1 / f(x)就没有意义了)

加以说明。
